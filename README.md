# Recursion Blockchain Analysis (Static Prototype)
web: https://blockchain-git-main-kamyab-s-projects.vercel.app?_vercel_share=EGWWTIxVSk5qZvQNHzakNJHLGQRbTuIG

A static, responsive dashboard prototype with animated background, 6 sample asset cards, live stats simulation, and a hash validator.

## Files
- `index.html` — main HTML
- `styles.css` — styling
- `app.js` — interactive logic and animations
- `.gitignore`
- `README.md`

## How to use
1. Open `index.html` in a browser (Chrome/Edge/Firefox).
2. Edit `assets` array in `app.js` to adjust the cards.
3. Push the repo to GitHub, enable GitHub Pages (see instructions).
